renderer

  • function
can-ssr.renderer  
  • source

A renderer function used to generate html from a route.

render(route) 

render("/orders").then(function(html){
    res.send(html);
});

Parameters

  1. route {String}

    A route for which the application should render. For example "/orders".

Returns

{Promise<String>}

A promise that, when resolved, will contain the string representation of the page after rendering.

render(state)

Parameters

  1. state {Object}

    Instance data that will be passed into the [AppMap] constructor to use as the template's viewModel.

Returns

{Promise<String>}

A promise that, when resolved, will contain the string representation of the page after rendering.
