inline-cache

  • typedef

{Object}

 
  • source

A representation of page data that was used to render the page.

INLINE_CACHE={..}

A global object added to the page that contains all page data used to render the route.

Use

You can add the inline cache to your page using the asset helper:

<body>
 ...

 {{asset "inline-cache"}}

</body>

Will generate:

<body>
 ...

 <script>
    INLINE_CACHE = {"key": {...}}
 </script>

</body>
