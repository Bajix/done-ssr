html5shiv

  • function
 
  • source

Conditionally load html5shiv and register custom elements for use with legacy versions of Internet Explorer.

{{asset "html5shiv"}}

Creates script tags adding html5shiv and registering custom elements.

Use

html5shiv needs to be added to the page before any custom elements appear. It is best to place it in the <head>, probably after your css. Your page might look like:

<html>
<head>
    <title>My App</title>

    {{asset "css"}}
    {{asset "html5shiv"}]
</head>
<body>
    ...
</body>
</html>

This will produce something like:

<html>
<head>
    <title>My App</title>

    <link href="foo.css">
    <script src="path/to/html5shiv.js"></script>
    <script>
        html5.elements = "order-history can-import";
        html5.shivDocument();
    </script>
</head>
<body>
    ...
</body>
</html>
