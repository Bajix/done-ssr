asset

  • function
can-ssr.helpers.asset  
  • source

A can.stache helper that is used to add a type of asset to the page.

{{asset type}}

Add the type of asset to the page.

Parameters

  1. type {String}

    a type of asset such as css.

Returns

{DocumentFragment}

A fragment appended to the page in the location of the helper.

Use

Plugins can implement the [asset.register] interface to provide types of assets that might need to be added to the page.

For example the done-css plugin registers itself as a handler for css assets. This allows you to do:

<head>
    \{{asset "css"}}
</head>

which will generate:

<head>
    <style>
        body {
          ...
        }
    </style>
</head>

in development and in production:

<head>
    <link rel="stylesheet" href="/dist/bundles/index.css">
</head>
