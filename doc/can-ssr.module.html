Create a renderer that, when called, renders the page for a given route.

canSsr(config) 

var canSsr = require("can-ssr");

var render = canSsr({
    config: __dirname + "/package.json!npm"
});

Parameters

  1. config {Object}Optional

    Configuration options that are a SystemConfig.

Returns

{renderer(route)}

A renderer function that can be used to render a given route.
